## 我的程序

```c
Position BinarySearch( List L, ElementType X )
{
    Position left, right;
    left = 1;
    right = L->Last;
    while(left <= right)
    {
        Position middle = (left + right) / 2;    // 记录中间位置
        if(L->Data[middle] == X)return middle;
        else if(L->Data[middle] < X)left = middle +1;
        else if(L->Data[middle] > X)right = middle - 1;
    }
    return NotFound;
}
```
